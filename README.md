# Algoritmos em Grafos 📊

Repositório contendo implementações de algoritmos fundamentais em teoria dos grafos, desenvolvido para estudos académicos na UNIFEI.

## 📑 Estrutura do Projeto

### 1. **Introdução a Grafos**
Conceitos básicos e primeiros passos com grafos.
- `ex1.py` - Exercício básico de introdução a grafos
- `ex2.py` - Exercício intermediário
- `ex3.py` - Exercício avançado de introdução

### 2. **Matriz de Adjacência**
Implementações usando matriz de adjacência como estrutura de dados.
- `ex1.py` - Criar matriz de adjacência básica
- `ex2.py` - Operações em matriz de adjacência
- `ex3.py` - Consultas e modificações
- `ex4.py` - Cálculo de propriedades
- `ex5.py` - Problemas avançados com matriz
- `ex6.py` - Aplicações práticas
- `ex7.py` - Casos de uso complexos

### 3. **Lista de Adjacência**
Implementações usando lista de adjacência como estrutura de dados.
- `ex1.py` - Criar lista de adjacência básica
- `ex2.py` - Operações em lista de adjacência
- `ex3.py` - Consultas e modificações
- `ex4.py` - Cálculo de propriedades
- `ex5.py` - Problemas avançados
- `ex6.py` - Aplicações práticas
- `ex7.py` - Conversão entre estruturas
- `ex8.py` - Casos de uso complexos

### 4. **Busca em Grafos**
Algoritmos principais de busca e navegação.
- `BFS.py` - Busca em Largura (Breadth-First Search)
- `DFS.py` - Busca em Profundidade Recursiva (Depth-First Search)
- `DFSIterativo.py` - Busca em Profundidade Iterativa

### 5. **Ordenação Topológica**
Algoritmos para ordenação topológica em grafos acíclicos.
- `ordenacaoTopotologica.py` - Implementação principal da ordenação topológica
- `TempoVertices.py` - Análise de tempo e processamento de vértices

### 6. **Percurso e Conectividade**
Algoritmos para análise de conectividade e percursos especiais.
- `Alcancabilidade(Warshall).py` - Algoritmo de Warshall para atingibilidade
- `CircuitoEuleriano.py` - Detecção e construção de circuitos Eulerianos

## 🚀 Como Usar

### Pré-requisitos
- Python 3.6+
- Nenhuma dependência externa

### Executar um exemplo
```bash
python "Lista De Adjacencia/ex1.py"
```

### Executar um algoritmo específico
```bash
python "Busca em Grafos/BFS.py"
```

## 📚 Conceitos Abordados

- ✅ Representação de grafos (matriz e lista de adjacência)
- ✅ Busca em largura (BFS)
- ✅ Busca em profundidade (DFS) - recursiva e iterativa
- ✅ Ordenação topológica
- ✅ Conectividade e componentes conexas
- ✅ Alcançabilidade entre vértices (Algoritmo de Warshall)
- ✅ Circuitos Eulerianos
- ✅ Análise de propriedades de grafos

## 🎯 Objetivo

Este repositório é um material de estudo para a disciplina de **Algoritmos em Grafos** do 3º Período do curso na UNIFEI, com foco em:
- Compreender estruturas fundamentais de grafos
- Implementar algoritmos clássicos
- Aplicar conhecimento em problemas práticos

## 📝 Notas

Todos os scripts são implementados em Python puro, sem dependências externas, facilitando o aprendizado dos conceitos fundamentais sem abstrações de bibliotecas de alto nível.
